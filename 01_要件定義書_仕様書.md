## 1. プロジェクト概要

### 1.1 背景
- 海外移住・就職・転職を検討する人は年々増加しているが、「移住後に自分の資産がどうなるか」を具体的にシミュレーションできるツールが存在しない
- Excelで個人的に試算するケースが多いが、為替リスクやインフレ考慮、複数シナリオ比較は困難
- 既存の生活費比較サイト（Numbeoなど）は現時点の比較のみで、将来予測機能がない
- 海外移住は人生の大きな決断であり、データに基づいた意思決定のニーズが高い

### 1.2 コンセプト
**「もしあなたがこの都市に移住したら、5年後どうなる？」**

旅行者向けではなく、以下を真剣に検討している人向けの資産シミュレーションプラットフォーム：
- 海外就職
- 海外転職
- 駐在
- ワーキングホリデー
- 留学
- 永住

### 1.3 ターゲットユーザー

| ユーザー種別 | 詳細 |
|---|---|
| **海外就職検討者** | 現在の勤務国から他国への転職を検討している社会人 |
| **海外移住検討者** | 永住・長期滞在を考えている個人・家族 |
| **ワーホリ検討者** | ワーキングホリデーで渡航予定の若年層 |
| **留学検討者** | 留学中・後の生活費・資産計画を立てたい学生 |
| **デジタルノマド** | リモートワークで最適な居住地を探している人 |
| **FIRE志向者** | 早期退職を目指し、最適な居住国を検討している人 |

### 1.4 対応言語
- 英語（主言語 - グローバルターゲット）
- 日本語
- 将来：中国語、韓国語、スペイン語

### 1.5 差別化ポイント

| 既存サービス | 限界 | MoveWorthの優位性 |
|---|---|---|
| Numbeo | 現時点のスナップショットのみ | 5年間の将来予測・資産推移 |
| Excel個人試算 | 単年・静的・共有不可 | 複数シナリオ同時比較・可視化 |
| 一般的な家計簿アプリ | 国内のみ・過去データ | 国際比較・将来予測・リスク分析 |
| 為替計算アプリ | 単純換算のみ | 為替変動リスクの確率分布 |

### 3.2 計算ロジック

#### 3.2.1 年間貯蓄額
```
年間貯蓄額 = 年収 × (1 - 実効税率) - (家賃 × 12) - (生活費 × 12)
```

#### 3.2.2 N年後の資産（基本）
```
資産(n) = 資産(n-1) × (1 + 投資利回り) + 年間貯蓄額(n)
年間貯蓄額(n) = 年収 × (1 + 昇給率)^n × (1 - 実効税率)
              - (家賃 × (1 + インフレ率)^n × 12)
              - (生活費 × (1 + インフレ率)^n × 12)
```

#### 3.2.3 為替考慮後の資産比較
```
移住先資産（基準通貨換算） = 移住先資産(n) / 為替レート(n)
為替レート(n) = 現在の為替レート × (1 + 為替変動率)^n

※ 為替変動率のデフォルト = 0（ユーザー指定可能）
※ モンテカルロでは過去のボラティリティを使用
```

#### 3.2.4 実質資産（インフレ考慮後）
```
実質資産(n) = 名目資産(n) / (1 + 基準国インフレ率)^n
```

#### 3.2.5 FIRE年齢計算
```
FIRE必要資産 = 年間支出 × 25（4%ルール）
年間支出 = (家賃 + 生活費) × 12 × (1 + インフレ率)^n

FIRE達成年 = 資産(n) ≥ FIRE必要資産(n) を満たす最小のn
FIRE年齢 = 現在年齢 + FIRE達成年
```

#### 3.2.6 モンテカルロシミュレーション（Premium）
```
各試行 i (i = 1 to 1000):
  各年 n:
    為替レート(n,i) = 為替レート(n-1,i) × exp(μ_fx - σ_fx²/2 + σ_fx × Z)
    投資リターン(n,i) = 期待リターン + σ_inv × Z
    インフレ(n,i) = 基本インフレ率 + σ_inf × Z

    ※ Z = 標準正規乱数
    ※ μ_fx = 為替のドリフト率（過去データから推定）
    ※ σ_fx = 為替のボラティリティ（過去データから推定）

結果:
  P10（悲観）、P50（中央）、P90（楽観）の資産推移を表示
```

#### 3.2.7 感度分析（Premium）
```
各パラメータ p を ±10%, ±20% 変動させた場合の
最終資産の変化量を算出し、トルネードチャートで表示

対象パラメータ:
  - 為替レート
  - 昇給率
  - インフレ率
  - 投資利回り
  - 税率
```
